سياسة الخصوصية لتطبيق محوّل التاريخ (Calendar Converter)

الإصدار: 2.0.0 | تاريخ التحديث: 31 مايو 2026
اسم الحزمة البرمجية: com.dbs.dateconverter

1. عام وهوية التطبيق

لا يتطلب هذا التطبيق تسجيل دخول أو إنشاء حساب. نحن لا نجمع بياناتك الشخصية مثل الاسم، البريد الإلكتروني، الهاتف، أو الموقع الجغرافي الدقيق (GPS).

2. البيانات المخزنة محلياً (SharedPreferences)

تُخزن البيانات التالية على جهازك فقط ولا تُرفع إلى خوادمنا أبداً: اختيار اللغة (14 لغة)، تفضيل الثيم (فاتح/داكن)، كاش تقويم المناسبات، وعدادات طلب التقييم.

3. خدمات Firebase (Android فقط)

يستخدم التطبيق Firebase Analytics لمتابعة الشاشات المفتوحة (مثل converter_tab)، و Crashlytics لإرسال سجلات الأعطال، و Remote Config لجلب الرابط الفعلي لسياسة الخصوصية. يتم تعطيل الجمع في وضع التطوير (Debug).

4. الإعلانات — Google AdMob

يعرض التطبيق إعلانات (App Open, Interstitial, Native, Rewarded). قد تجمع Google معرف الإعلانات والـ IP ونوع الجهاز. نستخدم منصة UMP لطلب موافقة الإعلانات المخصصة وفقاً لقوانين GDPR.

5. خدمات الطرف الثالث والإشعارات

يتكامل التطبيق مع AlAdhan API لجلب التقاويم و Google Fonts لتحميل الخطوط. الإشعارات المحلية تُجدول محلياً بناءً على منطقة الجهاز الزمنية (وليس GPS). التحويل الأساسي يعمل بدون إنترنت لوجود ملف embedded_occasions.json مدمج بالتطبيّق.

6. حقوق المستخدم والتواصل

يمكنك حذف بياناتك المحلية بمسح كاش التطبيق من إعدادات الهاتف. للتواصل مع المطور: naram3096@gmail.com.

Privacy Policy - Calendar Converter

Version: 2.0.0 | Updated: May 31, 2026
Package Name: com.dbs.dateconverter

1. General & Identity

No login or registration required. We do NOT collect personal info such as names, emails, phone numbers, or precise GPS data.

2. On-Device Storage (SharedPreferences)

The following are stored locally on your device and never uploaded: language settings (14 languages), theme choices (Light/Dark/Auto), cached occasions, and review request counters.

3. Firebase Services (Android Only)

Implements Firebase Analytics (monitors screens like converter_tab, settings), Crashlytics (logs bug stack traces), and Remote Config to fetch policy assets. Diagnostics are deactivated in Debug mode.

4. Advertisements — Google AdMob

Displays App Open, Interstitial, Rewarded Interstitial, and Native Ads. Google gathers Ad ID, IP address, and hardware parameters. Consent for personalized ads is managed using the Google UMP SDK (GDPR compliant).

5. Permissions & Connectivity

Utilizes AlAdhan API and Google Fonts. Local notifications are scheduled via system timezone boundaries, requiring POST_NOTIFICATIONS and exact alarm parameters. Conversion utilizes embedded_occasions.json fully offline. Contact: naram3096@gmail.com.
